## Task

List all React component files (.js, .jsx, .ts, .tsx) under `components/`.

If a subdirectory is given as an argument (e.g. `appointments`, `auth`, `ui`),
list only that one.

## Output Format

- Numbered list of files with their relative paths
- Brief one-line description of each (infer from filename)
- Summary count at the end

If no files found, output "No components found."
