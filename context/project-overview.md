# Project Overview

`Appointment Manager` official name `Rendez-vous` is a French base Next.js web app that let user create, edit, update and delete (CRUD) an appointment.

## Authentication

User must log in with a google account to manage the appointement.

## Tech Stack

Tech Stack used or will be used in this project, depending on features, they will be list more...

```
- Next.js            # Framework
- TypeScript         # Typed programming language
- NextAuth           # Authentication
- Neon               # Database
- Prisma             # ORM
```
